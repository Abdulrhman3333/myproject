## üöÄ Step-by-Step: Deploy for FREE on Railway

### 1. Create Railroad Account
```
Go to: railway.app
Sign up with GitHub
```

### 2. Create New Project
```
Click "New Project"
Select "GitHub Repo" or "Empty Project"
```

### 3. Add Services
```
- Click "Add Service"
- Select "GitHub Repository"
- Choose your tahfeed repository
```

### 4. Configure Environment
```
Settings ‚Üí Variables
Add:
  SECRET_KEY=your-generated-key
  DEBUG=False
  ALLOWED_HOSTS=your-domain.com
  DB_NAME=tahfeed_db
  DB_USER=your_user
  DB_PASSWORD=your_password
```

### 5. Add Database
```
Click "Add"
Select "PostgreSQL"
Wait for database to deploy
```

### 6. Configure Database Connection
```
Railway auto-creates DATABASE_URL
Update your settings.py to use it
```

### 7. Deploy
```
Push to GitHub
Railway auto-deploys
Wait ~5 minutes
Visit your live site!
```
