## 👨‍🎓 للطلاب وأولياء الأمور

### تسجيل طالب جديد
1. افتح الصفحة الرئيسية: http://127.0.0.1:8000/
2. املأ جميع البيانات المطلوبة:
   - الاسم الثلاثي
   - رقم هوية الطالب (10 أرقام، يبدأ بـ 1 أو 2)
   - رقم هوية ولي الأمر
   - جوال الطالب (اختياري)
   - جوال ولي الأمر (مطلوب)
   - الصف الدراسي
   - تاريخ الميلاد
   - آخر جزء تم اختباره
   - التحفيظ السابق ( ان وُجد ) (اختياري)
   - الحي
3. اضغط "إرسال طلب التسجيل"
4. ستظهر رسالة نجاح وسيكون الطالب في قائمة الانتظار

---

## 👨‍🏫 للمعلمين

### تسجيل الدخول
1. اذهب إلى: http://127.0.0.1:8000/login/
2. أدخل اسم المستخدم وكلمة المرور
3. اضغط "تسجيل الدخول"

### التحضير اليومي
1. بعد تسجيل الدخول، اذهب إلى: http://127.0.0.1:8000/attendance/
2. سترى قائمة بطلابك المنتظمين فقط
3. كل طالب يظهر مع زر الحالة (افتراضياً: حاضر - أزرق)
4. اضغط على الزر لتغيير الحالة:
   - حاضر (أزرق)
   - غائب (أحمر)
   - مستأذن (أصفر)
5. بعد الانتهاء، اضغط "إتمام التحضير وحفظ النتائج"

### ملاحظات للمعلمين:
- يمكنك فقط رؤية طلابك المخصصين لك
- لن ترى الطلاب المنتظرين (هذه مهمة المدير)
- التحضير محمي من التكرار - سجل واحد لكل طالب في اليوم

## 👨‍💼 للإداريين (المدير)

### تسجيل الدخول كمدير
- استخدم حساب السوبر يوزر الذي أنشأته
- أو استخدم لوحة Django Admin: http://127.0.0.1:8000/admin/

### قبول الطلاب المنتظرين
1. اذهب إلى: http://127.0.0.1:8000/pending/
2. سترى قائمة بجميع الطلاب في حالة "منتظر"
3. لكل طالب معلومات: الاسم، الصف، آخر جزء مختبر
4. اضغط "قبول الطالب" لتحويل حالته إلى "منتظم"
5. بعد القبول، يجب تعيين معلم للطالب من لوحة Admin

### تعيين المعلمين للطلاب
1. اذهب إلى لوحة الإدارة: http://127.0.0.1:8000/admin/
2. اختر "Students"
3. اختر الطالب الذي تريد تعيين معلم له
4. من قائمة "المعلم المسؤول"، اختر المعلم المناسب
5. احفظ التغييرات

### إدارة الطلاب من لوحة Admin
يمكنك:
- تعديل بيانات الطالب
- تغيير حالة الطالب
- تعيين/تغيير المعلم
- حذف طالب
- البحث بالاسم أو رقم هوية الطالب
- تصفية حسب: الحالة، المرحلة، المعلم، الصف

### عرض سجلات الحضور
1. في لوحة Admin، اختر "Attendances"
2. يمكنك:
   - عرض جميع سجلات الحضور
   - تصفية حسب التاريخ أو الحالة
   - البحث باسم الطالب
   - التصفح حسب التاريخ (Calendar navigation)

## 🧪 البيانات التجريبية

### الطلاب التجريبيون:
1. عبدالله محمد الغامدي - خامس ابتدائي - منتظم (معلم: أحمد)
2. خالد سعد الشهري - ثالث متوسط - منتظم (معلم: أحمد)
3. فيصل أحمد الزهراني - ثاني ثانوي - منتظم (معلم: محمد)
4. عمر ناصر العنزي - أول ابتدائي - منتظم (معلم: محمد)
5. سلطان عبدالرحمن الدوسري - رابع ابتدائي - **منتظر**
6. ياسر فهد المطيري - أول متوسط - **منتظر**
7. راشد عبدالعزيز الحربي - ثالث ثانوي - منتظم (معلم: محمد)
8. طلال حمد السليمان - ثاني ابتدائي - **منتظر**

### حسابات المعلمين:
```
معلم 1:
Username: معلم_احمد
Password: teacher123
الطلاب: عبدالله، خالد

معلم 2:
Username: معلم_محمد
Password: teacher123
الطلاب: فيصل، عمر، راشد
```

## 🔄 سير العمل الكامل

### 1. تسجيل طالب جديد
- يدخل ولي الأمر البيانات من الصفحة الرئيسية
- الطالب يُضاف بحالة "منتظر"

### 2. مراجعة وقبول
- المدير يراجع الطلبات في صفحة "الطلاب المنتظرين"
- يقبل الطالب (تتغير الحالة إلى "منتظم")
- يعين له معلماً من لوحة Admin

### 3. التحضير اليومي
- المعلم يسجل دخوله
- يفتح صفحة التحضير
- يسجل حضور/غياب طلابه
- يحفظ النتائج

### 4. المتابعة
- المدير يمكنه مراجعة سجلات الحضور من لوحة Admin
- عرض التقارير والإحصائيات
- متابعة أداء الطلاب

## ⚙️ إعدادات النظام

### المراحل الدراسية (تُحسب تلقائياً):
- **مبكرة:** أولى - ثانية - ثالثة ابتدائي
- **عليا:** رابعة - خامسة - سادسة ابتدائي
- **متوسط:** أول - ثاني - ثالث متوسط
- **ثانوي:** أول - ثاني - ثالث ثانوي
- **جامعي:** الدراسة الجامعية

### حالات الطالب:
- **منتظر:** في قائمة الانتظار (لم يُقبل بعد)
- **منتظم:** مقبول ولديه معلم مخصص

### حالات الحضور:
- **حاضر:** الطالب حاضر
- **غائب:** الطالب غائب بدون عذر
- **مستأذن:** الطالب غائب بعذر

## 🚀 نصائح وإرشادات

### للمعلمين:
✅ سجل التحضير يومياً في بداية الحلقة
✅ تأكد من حفظ النتائج قبل إغلاق الصفحة
✅ يمكنك تغيير حالة الطالب عدة مرات قبل الحفظ

### للإداريين:
✅ راجع قائمة المنتظرين بشكل دوري
✅ وزع الطلاب بالتساوي على المعلمين
✅ راجع سجلات الحضور لمتابعة التزام الطلاب
✅ استخدم فلاتر البحث للوصول السريع للبيانات

### أمنية:
🔐 غيّر كلمات المرور الافتراضية فوراً
🔐 لا تشارك بيانات الدخول
🔐 اعمل نسخة احتياطية من قاعدة البيانات بانتظام
